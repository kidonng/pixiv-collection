# pixiv Collection

Dead simple pixiv collection showcase. Built with Vuetify 2.x and Photoswipe.

## Usage

[Put your config & collection](./config.js) and that's all.

- Setup `yarn`
- Development `yarn serve`
- Build `yarn build`
